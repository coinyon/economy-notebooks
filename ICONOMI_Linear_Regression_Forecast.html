ICONOMI Linear Regression Forecast¶

This notebook performs a linear regression forecast on the ICONOMI quarterly report data.

Please note that a linear regression on this kind of data is not useful for forecasting future values, because the evolution of a company is inherentely non-linear. I primarily created this notebook to get to know sklearn and sklearn_pandas.

Create a linear regression and predict future values¶

A linear regression works by fitting model parameters to given data. To fit the data, we need to scale (or normalize) it first. In the following lines, the a DataFrameMapper is created with a StandardScaler for every column.

In [4]:
cols = df.columns[1:]
std_scaler = sklearn.preprocessing.StandardScaler()
mapper = DataFrameMapper([(cols, std_scaler)])

We can now use a LinearRegression to fit our data:

In [5]:
lm = sklearn.linear_model.LinearRegression()
lm.fit(np.array([df.index]).T, mapper.fit_transform(df.copy()));

def num_to_quarter(q, pos=None):

How long ICONOMI can burn funds¶

Next up, I'll try to get a rough estimate how long ICONOMI can remain operational without requiring new capital if they keep burning money at the current burn rate. For this, we simply divide the latest book value by the latest burn rate:

In [9]:
latest = df.iloc[df.index.max()]
book_value = latest["Book Value"]
burn_rate = latest["Expenditures"] - latest["Revenue"]
book_value / burn_rate / 4
Out[9]:
6.574715078741297

So, ICONOMI can keep doing this for 6.5 years, assuming that the market will stay the same and their burn rate will stay constant (both assumptions are nonsense, but it gives a rough idea).

What if the burn rate would increase to our prediction about? To figure that out, we will run an expanding() window function over our burn rate predictions and accumulate it pair-wise. Then we find the first row where this accumulation is above the book value.

In [10]:
burn_rate = predict_df["Expenditures"] - predict_df["Revenue"]
(predict_df[burn_rate.expanding().sum() > book_value].index[0] - df.index.max()) / 4
Out[10]:
3.75

Assuming that the burn rate continues to increase, ICONOMI has about 3.75 years left.
